Mock LC Parser

Description

Plugin class: Analyzer

This plugin generates and parses report files as if they were generated from a liquid chromatograph (LC). The data generated is random and this plugin is only meant to be used as a demonstration of a file parser as an analyzer plugin.

Acquisition

Download and install the plugin and required drivers from the official GitHub release pages listed below.

Downloads

  • Plugin (.vip file): Mock LC Parser Plugin – Releases

  • Drivers: This plugin does not require any additional drivers.
